# Azure AI Automation Mini Project

A FastAPI-based backend deployed on Azure App Service, designed to explore
cloud-hosted AI automation workflows and API-driven integration patterns.

## Tech Stack

- Python
- FastAPI
- Azure App Service
- ZIP Deployment
- GitHub (CI/CD planned)

## Endpoints

- GET /health — service health check
- GET /ready — readiness probe

## Deployment

Deployed to Azure App Service (Linux) using ZIP deploy with a custom startup script.

## Status

Infrastructure and deployment complete. AI automation logic in progress.
